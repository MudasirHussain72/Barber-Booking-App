# barbar_booking_app

A Barber Booking App Flutter project.
# Features
* Nearby Shops Data
* Firebase Cloud Push Notifications
* Google Map Current Location
* Form Validation
* Firebase Authentication
* Realtime Booking Changes
* Firebase Cloud Firestore Database
